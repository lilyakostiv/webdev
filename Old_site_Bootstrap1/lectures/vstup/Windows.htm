Створення об’єктів Windows

Основні типи об’єктів в ОС Wіndows – це папки (каталоги), документи (файли) і вказівники (ярлики).

Створювати об’єкти можна двома способами: користуватись меню Файл відкритого вікна або контексним меню, яке викликається натискуванням правої кнопки миші.

Об’єкти можуть створюватись у будь-якому вікні, яке має меню Файл з командою Создать, для цього часто використовують вікна  програми Проводник або Мой компьютер. На робочому столі використовується контексне меню. При виборі команди Создать (меню Файл або контексного меню) відкривається підменю, у якому подано перелік типів об’єктів, які можна створити у відкритому вікні або на робочому столі. За вибору рядка Папка або будь-якого рядка, розташованого нижче роздільної лінії, у вікні або на робочому столі з’являється нова піктограма, під якою розміщено затемнене поле, у яке потрібно ввести з клавіатури ім’я нового об’єкта (папки або документа).

Тобто в ОС Windows документ може бути створений без запуску будь-яких програм. У цьому випадку йдеться про відкриття і надання імені деякому порожньому файлу.

Переіменування об’єктів здійснюється в режимі безпосереднього редагування. Для включення цього режиму треба один раз клацнути по значку ярлика, а другий раз – по полю його імені. При цьому текст обрамляється, фон поля тексту імені стає чорним, а вказівка набирає вигляду вертикальної риски. Вказівку миші необхідно встановити у потрібному місці рамки і клацнути лівою кнопкою; після цього можна починати процес редагування.

 

Копіювання і переміщення об’єктів

У ОС Windows операції копіювання і переміщеня можна виконувати  кількома способами:

-          використовуючи команди меню вікна Правка (можна користуватись “гарячими” клавішами);

-          за допомогою команд контекстного меню;

-          використовуючи метод перетягування значків за допомогою миші.

Для копіювання і переміщення об’єктів використовується буфер обміну – деяка
область  пам’яті комп’ютера, призначена для тимчасового зберігання інформації. Буфер обміну використовують для обміну інформацією між відкритими додатками. Інформація у буфер обміну записується за допомогою команди Копировать (гарячі клавіші Ctrl+C) або Вырезать (гарячі клавіші Ctrl+X) меню Правка, а зчитується з нього за допомогою команди Вставить (гарячі клавіші Ctrl+V) цього самого меню.

Команду Вырезать потрібно використовувати тільки в разі переміщення об’єктів. При переміщенні об’єкта він переноситься у нове місце. При копіюванні на новому місці створюється тільки копія початкового об’єкта, а на старому місці він залишається без змін.

 

Копіювання об’єкта  за допомогою миші виконують таким чином:

-          вибрати об’єкт, який потрібно копіювати (клацнути по значку мишею);

-          натиснути клавішу Ctrl і, не відпускаючи її, сумістити вказівку миші з піктограмою  вибраного об’єкта та натиснути ліву кнопку миші;

-          не відпускаючи ліву кнопку миші, а також клавішу Ctrl, перетягнути копію піктограми вибраного об’єкта в потрібне місце і відпустити ліву кнопку миші та клавішу Ctrl.

Для виконання операції переміщення потрібно вибрати об’єкт і перетягнути його у потрібну папку.

 

 При переміщенні об’єктів у папки, що знаходяться на іншому диску, потрібно утримувати клавішу Alt. А копіювання об’єктів між дисками здійснюється звичайним перетягуванням об’єкта мишею.

Допускається виконання операцій з групами об’єктів. Для цього їх необхідно вибрати (відмітити). Щоб відмітити групу файлів, розташованих у довільному порядку, необхідно клацнути по їх значках, утримуючи клавішу Ctrl, а якщо вони розміщені один за одним, - клавішу  Shift (клацнути на першому та останньому).

 

Знищення і відновлення об’єктів
        Непотрібні об’єкти треба знищувати. При спробі знищення об’єкти не знищуються, а вміщуються у папку Корзина. Це дозволяє відновити випадково знищені об’єкти. Якщо очистити вміст  Корзини, то відновлення стає неможливим.

Знищення об’єктів відбувається так:

·         вибрати об’єкт або групу об’єктів, які потрібно знищити;

·         відкрити меню Файл або викликати контексне меню і клацнути вказівкою миші по команді Удалить;

·         після появи діалогового вікна Подтверждение удаления папки натиснути кнопку Да для підтвердження знищення або Нет, якщо передумали знищувати об’єкт.

Також можна знищити відмічений об’єкт без використання меню, натискуванням клавіші Delete або використовуючи метод перетягування об’єкта в кошик.

Для відновлення об’єктів спочатку потрібно відкрити папку Корзина. Далі потрібно вибрати ті об’єкти, які необхідно відновити, відкрити меню Файл і вибрати рядок Восстановить. Вибрані об’єкти повертаються у те саме місце, звідки вони були знищені.

Відмічені об’єкти можуть бути переміщені (скопійовані) з кошика в будь-яку папку методом перетягування або через буфер обміну (командами Правка/Вырезать і Правка/Вставить).

Використовуючи меню Файл вікна Корзина, можна знищити з неї відмічені об’єкти (рядок Удалить) або повністю очистити її, для чого призначена команда Очистить корзину.

 

Створення вказівника (ярлика) на об’єкт

Для створення вказівника у контексному меню об’єкта вибрати команду Копировать, тоді у потрібній папці задати команду Вставити ярлик.

 

Завдання 1  ( Windows )

1.      Викликати головне меню Windows (кнопка Пуск).

2.      Виконати команду Найти, за допомогою якої знайти програму bp.exe.

3.      Змінити розміри та розміщення на екрані одержаного діалогового вікна Поиск.

4.      Збільшити це вікно на весь екран, потім зменшити до кнопки на панелі завдань.

5.      Відновити це вікно з панелі завдань, потім закрити.

6.      Виконати команду Справка головного меню, вибрати команду Указатель (зверху) і знайти інформацію про форматування дисків.

7.      За допомогою контексного меню робочого стола впорядкувати піктограми по назвах.

8.      Встановити автоматичний режим.

9.      Кошик перетягнути в кінець списку піктограм.

10.  Вийти з Windows (завершити роботу користувача user...).

 

Завдання 2 ( Windows )

1.      Створити папку з назвою Клас (наприклад, 8-Б) на диску U.

2.      У цій папці створити папку з назвою Учні   та   документ Word з назвою Математик.

3.      Документ Word Математик перемістити в папку Учні (перетягуванням).

4.      У папці Учні документ Математик переіменувати на Фізик і скопіювати в папку Клас (використати меню вікна).

5.      Знищити папку Учні і документ Фізик одночасно (використовуючи контексне меню), потім відновити її з Кошика.

6.      Створити Ярлик для папки Клас на робочому столі.

7.      Знищити папку Клас і ярлик на цю папку (шляхом перетягування в Кошик), потім очистити Кошик.
