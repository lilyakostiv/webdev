3. Умовний оператор

            У програмі оператори виконуються послідовно. Проте часто доводиться змінювати послідовний порядок виконання дій. У Паскалі, як і в інших мовах програмування, є кілька можливостей реалізації вибору. Для вибору однієї з двох альтернатив переважно використовують умовні оператори if, а для вибору з декількох можливих варіантів - оператор case.

Розглянемо умовні оператори. Є два види таких операторів - повний та скорочений. Повний оператор можна зобразити такою схемою:

           

If <умова> then <оператор 1> else <оператор 2>;

 

В якості <умови>  використовується логічний вираз, <оператор1> та <оператор2> - довільні оператори мови Паскаль. Якщо цей логічний вираз істинний, то виконується  <оператор 1> , а якщо хибний - виконуєтся <оператор 2>.

Якщо замість <оператора 1>  або <оператора 2> потрібно виконати більше ніж один оператор, тоді використовується складений оператор :

 

begin <оператор 1>;<оператор 2>;…;<оператор n> end;

 

Скорочений запис оператора розгалуження має вигляд:

 

If <умова> then <оператор 1>;

3.24. Якщо квадратне рівняння з коефіцієнтами a,b,c (a≠0) має дійсні корені, логічній змінній t присвоїти значення true, а змінним х1 та х2 – корені, інакше змінній t присвоїти false.

program exampl;

var a, b, c, d, x1,x2: real;

      t: boolean;

begin { розв'язування квадратного рівняння}

writeln('введіть значення коеф. квадратного рівняння a , b, c');

read (a,b,c);

d := sqr(b) - 4*a*c; {обчислення дискримінанта}

{перевірка дискримінанта та обчислення коренів}

if d<0 then t := false { дійсних коренів немає}

         else  begin { складений оператор для обчислення коренів та  виведення їх                                значень і присвоєння значення змінній t}

x1 := (-b+ sqrt(d))/(2*a);

x2 := (-b-sqrt(d))/(2*a);

writeln('x1=',x1,'x2=',x2);

t := true

                   end; {кінець умовного оператора}

writeln('t=',t)

end.

Виконавши цю програму на комп'ютері бачимо, що у випадку відсутності у квадратного рівняння коренів, на чистий екран буде виведено лише значення t=false. Якщо  рівняння має дійсні корені, то спочатку будуть виведені значення цих коренів, а пізніше значення t=true.

У наведеній програмі оператор виведення writeln('t=',t) виконується незалежно від значення умови в умовному операторі.
