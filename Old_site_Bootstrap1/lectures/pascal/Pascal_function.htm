11. Процедури-функції

Function <ім'я функції>(<список формальних параметрів>):<ім'я типу>;

У тілі функції обов'язково повинен бути оператор присвоєння виду:

<ім'я функції>:=<вираз>

 

Опишемо функцію f(n)=n!:

Function FACT(n:integer):integer;

Var i,s:integer;

Begin

 s:=1; 

 For i:=1 to n do s:=s*i

 FACT:=s

End;

 

Program PR;

Var p,p2,a:integer;

Function…

Begin

  p:=FACT(10); write(p);

  Read(a);

  p2:=FACT(a); write(p2);

 End.

 

Запишемо процедуру обчислення факторіала:

Procedure Factor(n:integer; var m:integer);

Var i:integer;

Begin

  m:=1;

  For i:=1 to n do m:=m*i;

End;

Тоді звертання до процедури обчислення факторіалу відбувається так: Factor(a,p);

 

Приклад. Обчислити y=k!+(3k)!-5!

Використання процедури: Factor(k,m1); Factor(3*k,m2); Factor(5,m3);  y:=m1+m2-m3;

Використання функції: y:=FACT(k)+FACT(3*k)-FACT(5);
