Цілочислові вирази

         До складу таких виразів можуть входити лише цілочислові константи, змінні та функції. Є дві арифметичні функції, які дають цілочисловий результат, якщо аргумент функції цілочисловий. Це функції

abs(x )             x

sqr(x)                 

            Цілочислові  належать до дискретних величин і тому для них визначено функції наступності, які встановлюють значення succ(x) – наступного  та pred(x) – попереднього елементів. У цьому випадку ці функції визначено так:

succ(x)  дорівнює  x + 1;

pred(x) )  дорівнює  x - 1.

            Над цілочисловими елементами визначено  операції  додавання +, віднімання -, множення *.

Для ділення використовують дві операції: div  - результатом є частка від ділення двох цілих чисел та mod - результатом є остача від ділення. Операції div та mod вимагають цілочислових аргументів.

Наприклад, після виконання таких операторів присвоєння (змінні i,j: integer) i:=10 div 3; j:=10 mod 3; змінні одержать такі результати i=3; j=1.  Використовуючи операції ділення відповідні службові слова повинні бути виділені з обох боків пропусками. Також треба пам'ятати про те, що використання символу ділення / перетворює вираз до дійсного типу.

            Розглянемо приклад програми, яка для цілого двоцифрового числа обчислює суму його цифр.

program  Examp1;

var c,c1,c2,s: integer;

begin {обчислення суми цифр двоцифрового числа}

            writeln(' введіть ціле двоцифрове число');        read(c);

            c1:= c div 10;                         {знаходження  першої цифри}

            c2:= c mod 10;                      {знаходження другої цифри}

            s:= c1 + c2;                           {обчислення суми цифр}

            writeln('сума цифр числа ', c, '  дорівнює  ',s);

end.

            Оператор присвоєння для цілочислових величин має вигляд

<цілочислова змінна> := <цілочисловий вираз>

            Для перетворення дійсного виразу до цілого типу треба використати одну з двох функцій trunc(x) або round(x),  які мають аргумент дійсного типу, а результат цілого типу.

            За функцією trunc(x) перетворення до цілого відбувається  шляхом відкидання дробової частини, наприклад trunc(6.9)=6 , trunc(6.2)= 6, trunc(-1.8)= -1. За функцією round(x) відбувається заокруглення до ближчого цілого, наприклад, round(6.9)=7 , round(6.2)=6 , round(-1.8)= -2 .

            Наступний приклад ілюструє використання функції перетворення типів.

Скласти програму, яка дає змогу визначати першу цифру дробової частини дійсного числа, заданого у формі з фіксованою крапкою.

program  Examp2;

var  x : real;

       c : integer;

begin   {визначення першої цифри  дробової частини числа}

            writeln(' введіть дійсне  число');      read(x);

c:= trunc(x * 10)  mod 10;  {знаходження першої цифри дробової частини}

            writeln('перша  цифра числа ', x, ' дорівнює  ',c);

end.
