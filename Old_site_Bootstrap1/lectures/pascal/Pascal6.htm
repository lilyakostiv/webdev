6. Перелічувані та обмежені типи

Оператор варіанту СASE

Case <вираз> of

 <значення 1>:<оператор1>;

 …

 <значення n>:<оператор n>;

 else <оператор n+1>;

 end;

В залежності від значення <виразу>  виконується відповідний оператор, наприклад, якщо <вираз> рівний <значенню 1>, то виконується <оператор1> і т.д., якщо вираз не є рівний ні одному із вказаних значень, то виконується <оператор n+1>.

Скорочений оператор варіанту не містить частини else. Якщо потрібно виконати більше ніж один оператор, то застосовуємо складений оператор.

Оператор Case зручно використати замість такого запису:

                                   if B1 then S1 else

                                                if B2 then S2 else

                                                  …

                                                     if Bk then Sk  - це дуже громіздкий запис. 

 

Приклад. Обчислити значення змінної m в залежності від того, яке значення набуває змінна i.

Оператор case виглядає так:

Case i of

1:m:=1;

2:m:=sqr(2);

3:m:=sqr(3)*3;

else m:=0

end;

Якщо потрібно виконати спільну дію для кількох значень, то ці значення записують через кому.

Case i of

1,3,7:m:=i;

4,6,8:m:=sin(i);

9..20:m:=2*cos(i)

end;

 

Приклад. Обчислити значення змінної f в залежності від того яке значення набуває вираз i^2+5.

Program R2;

Var i,m:integer;

Begin

  Read(i);

  Case sqr(i)+5 of

   1..5:m:=sqr(i);

   6,9:m:=i+3

   else m:=10

 end;

 writeln(‘m=’,m)

end.
