ЗАПИТИ В ACCESS

 

1. Створити простий запит (Запит1) за допомогою майстра. Задати поля: Прізвище, Імя, Оцінка з інформатики, клас; класний керівник. Результат запиту - це таблиця. Відкрити запит в режимі конструктора: впорядкувати по оцінці з інформатики, клас задати 11-Б.

 

2. Створити простий запит (Запит2) за допомогою конструктора. Задати поля: Прізвище, Імя,  клас; класна кімната. Впорядкувати запит по прізвищу, показати список учнів, у яких класна кімната від 100 до 200, а клас 11-А або 11-Б, прізвище починається на літеру А.

Підказка. Можна використовувати в умові: and, or, between a and b, шаблони A*, B???C* (* - багато символів, ?- 1 або 0 символів).

 

3. Створити підсумковий запит (Запит3)  за допомогою майстра. Задати поля: клас, оцінка з інформатики, згрупувати по полю клас, для поля оцінка з інформатики задати підсумкову (агрегуючу) функцію середнє арифметичне (можна задавати: сума, max, min, кількість). Немає бути зайвих полів.

- Відкрити запит в режимі конструктора: замість середнє арифметичне задати функцію суми. - Клас задати 11-Б.

- Видалити поле клас і знайти максимальну оцінку для всіх учнів зі списку.

 

МОВА ЗАПИТІВ SQL

 

SELECT список полів

FROM список таблиць зі зв'язком між ними

WHERE умови  

GROUP BY список полів

ORDER список полів
