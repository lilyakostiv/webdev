Програма 5 ( з використанням MainMenu)

 

 

По події FormShow задати значення фіксованим (сірим) коміркам (По рядку: Оцінки, Математика, Фізика, Інформатика, Середня оцінка, Максимальна оцінка, по стовпцю: задати прізвища двох учнів).

Можна використати перелічуваний тип:

            const m: array [1..6] of string = ('Оцінки','Математика', ...);  тоді StringGrid1.Cells[i,0] := m[i+1];

 

Пункт меню Робота з таблицею повинна містити команди Середня оцінка (знайти середні оцінки учнів і записати у відповідні комірки таблиці) , Максимальна оцінка (знайти максимальні оцінки учнів і записати у відповідні комірки таблиці).

Пункт меню Робота з полем Memo буде містити команди Оцінки 1 учня (записати в поле Memo прізвище 1 учня і його середню та максимальну оцінки) , Оцінки 2 учня (записати прізвище та оцінки 2 учня).

Пункт меню Вихід завершити роботу.

 

Додати на форму ще 5 компонент:

 

1.      SaveDialog – для створення текстового файлу.

2.      OpenDialog – для відкриття текстового файлу.

3.      Memo – для відображення даних з файлу.

4.      Button – для запису даних таблиці  у текстовий файл.

5.      Button – для считування  даних з текстового файлу у поле Memo.
