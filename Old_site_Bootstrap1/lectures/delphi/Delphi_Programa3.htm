Програма 3     Інформація про учня

 

 

Задати кольори формі та всім панелям.

Для ComboBox у властивості Items задати перелік з 5 імен.

Для LabeledEdit вибрати властивість EditLabel і задати Caption.

Для кнопки BitBtn задати назву кнопки (властивість Caption) та малюнок (властивість Glyph) завантажити з файлу .bmp.

 

По натисканні кнопки Підсумок у Memo вивести:

1. Прізвище, ім'я (ComboBox1.text) та по-батькові учня.

2. Клас.

3. Улюблені предмети ( if CheckBox1.Checked then Memo1.Lines.Add(CheckBox1.Caption); )

4. Середнє арифметичне значення оцінок учня.

5. Оцінити досягнення учня (10..12 - відмінно, 7..9 - добре, 4..6 - задовільно, 1..3 - погано).
