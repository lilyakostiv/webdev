Контрольна робота по темі «Спеціальні типи даних у Паскалі”

 

I варіант

1. (5 балів) Ввести текст із малих латинських літер. Надрукувати всі літери, що входять у текст по одному разу.

 

2.  (7 балів) Задати список з 20 студентів, для кожного ввести такі дані: прізвище, номер групи, номер заліковки, оцінки з 10 предметів.

- Знайти кількість студентів, прізвища яких починаються на літеру К.

- Знайти середній бал кожного студента.

- Вивести прізвище та номер групи студента із номером заліковки 1000.

 

 

II варіант

1. (5 балів) Ввести ціле число. Знайти кількість цифр числа, які зустрічаються по одному разу.

 

2.  (7 балів) Задано список з 25 учнів, для кожного учня ввести прізвище, зріст, вагу, клас, школу.

- Знайти кількість десятикласників.

- Середню вагу учнів.

- Вивести прізвище найвищого учня.
